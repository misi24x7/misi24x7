Get SmartOps

🚀 Get SmartOps Running in Your Cluster

Deploy SmartOps in minutes with our step-by-step implementation guide

1

Prerequisites

  • Kubernetes cluster (v1.20+)
  • kubectl configured
  • Python 3.8+ (optional for local development)
  • Git access
2

Quick Deploy

Deploy with one command 
kubectl apply -f https://raw.githubusercontent.com/misi24x7/smartops/main/k8s/
3

Access Dashboard

  • Port-forward: kubectl port-forward -n smartops svc/smartops-dashboard 8501:8501
  • Open: http://localhost:8501
  • Start monitoring your cluster!

🎯 Deployment Options

🚁 Helm Chart Deployment

Add repository and install 
helm repo add smartops https://misi24x7.github.io/smartops
helm repo update
helm install smartops smartops/smartops --namespace smartops --create-namespace

Perfect for production deployments with easy upgrades and configuration management.

⚡ Direct Kubectl Deployment

Deploy from GitHub manifests 
git clone https://github.com/misi24x7/smartops.git
cd smartops/k8s
kubectl apply -f namespace.yaml
kubectl apply -f deployment.yaml
kubectl apply -f service.yaml

Full control over deployment with custom manifests and configurations.

🐳 Docker Compose Deployment

Run locally with Docker 
git clone https://github.com/misi24x7/smartops.git
cd smartops
docker-compose up -d

Quick local testing and development without Kubernetes cluster setup.

💻 Local Development Setup

Run from source 
git clone https://github.com/misi24x7/smartops.git
cd smartops/dashboard
pip install -r requirements.txt
streamlit run streamlit_app.py

Develop and customize SmartOps features locally with hot reloading.

⚙️ Configuration

Environment Variables

KUBECONFIG=/path/to/kubeconfig
TELEGRAM_BOT_TOKEN=your_bot_token
ANOMALY_SENSITIVITY=0.1
LOG_LEVEL=INFO

RBAC Configuration

apiVersion: rbac.authorization.k8s.io/v1
kind: ClusterRole
metadata:
  name: smartops-role
rules:
- apiGroups: [""]
  resources: ["pods", "services", "nodes"]
  verbs: ["get", "list", "watch"]

Persistent Storage

apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: smartops-storage
spec:
  accessModes: [ReadWriteOnce]
  resources:
    requests:
      storage: 10Gi
